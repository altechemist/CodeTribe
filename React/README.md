# React Introductory Projects

This repository hosts multiple projects showcasing a variety of functionalities, from CRUD operations to app development scripts. Below is a description of each project.

---

### **1. Employee Data**  
**Description**:  
A backend API for managing employee records using CRUD operations. Includes support for authentication and image uploads with Firebase.  

**Key Features**:  
- Add, edit, retrieve, and delete employee records.  
- Firebase integration for authentication and storage.  

**Directory**: `employee-data`  

---

### **2. Flavour Book**  
**Description**:  
A recipe management application that allows users to organize and manage recipes.  

**Key Features**:  
- Add, edit, and delete recipes.  
- User-friendly interface for browsing and searching recipes.  

**Directory**: `flavour-book`  

---

### **3. Insight**  
**Description**:  
A news app that keeps users informed with the latest articles.  

**Key Features**:  
- Bookmark news articles for later reading.  
- Offline reading support for saved articles.  

**Directory**: `insight`  

---

### **4. Luxe Hotel**  
**Description**:  
A hotel management system for booking and managing reservations.  

**Key Features**:  
- View and book hotel rooms with PayPal API integration.  
- Admin dashboard for adding and editing rooms, managing reservations.  

**Directory**: `luxe-hotel`  

---

### **5. Shopmate**  
**Description**:  
An application to create and manage sharable shopping lists.  

**Key Features**:  
- Collaborative list creation and sharing.  
- Intuitive user interface.  

**Directory**: `shopmate`  

---

### **6. Task Master**  
**Description**:  
A to-do application designed for productivity and task organization.  

**Key Features**:  
- Create, edit, and track daily tasks.  
- Manage deadlines and priorities effectively.  

**Directory**: `task-master`  

---

### **7. Create-Vite**  
**Description**:  
A script for quickly scaffolding a Vite-based application with Bootstrap pre-configured.  

**Key Features**:  
- Automates the setup process for Vite and Bootstrap.  
- Ideal for fast prototyping and development.  

**Directory**: `create-vite`  

---

## How to Use  

1. **Clone the Repository**:  
   ```bash
   git clone <repository-url>
   cd <project-name>
   ```

2. **Navigate to the Desired Project**:  
   Each project resides in its own directory. Choose the one you want to explore.  

3. **Install Dependencies**:  
   Use `npm install` or `yarn` in the project directory to install required packages.  

4. **Run the Project**:  
   Refer to the README file inside each directory for specific setup and usage instructions.
